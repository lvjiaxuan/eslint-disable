## Features


- [x] Based on ESLint's results from current file.
- [x] Select one or **multiple lines**, and disable it with:
  - [x] The editor context menu.
  - [x] Cmd(`eslint-disable.disableIT`).
  - [x] Keyboard shortcuts(`ctrl + alt + d`).
